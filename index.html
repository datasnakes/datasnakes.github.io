Apps

Here are a few apps we've created or are hosting on our server.

GPCR App

This app is our first major release. It helps to visualize the genetic data produced from orthologous alignments/phylogenetic analysis of GPCR and related genes.

Hall App

The Hall Project App is a meta analysis using the tools from the GPCR Orthologs Project. One of our collaborators took an interest in the APOL-1 gene, and we used the tools from our GPCR app to fetch data/analysis. This app is still under development.

Public FTP

Check out this repository which features our datasets and source code in downloadable chunks.



Try Markdown

This app allows users to interactively view, edit, and save Markdown formatted text. It's not an original app, but we have edited for our use and public use.

Genetics Metrics

The Genetics Metrics application is a R-shiny app created by the Oregon National Primate Research Center. It was created for individuals to host/visualize genetics data locally.

Phyloseq App

Shiny-phyloseq is an interactive web application that provides a graphical user interface to the microbiome analysis package for R, called phyloseq.
