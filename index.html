What We Do

We snake through difficult bioinformatics problems using Python and R programming. Go figure! Mainly, our focus is on creating Rshiny apps, python libraries, and pipelines that aid in bioinformatics projects.


Here are some of the projects we tackle:

  • RNASeq Analysis & Visualization
  • Orthology Inference
  • Microbiome Analysis & Visualization

Github Projects

Here are a few of our github projects.

cookiecutter-orthoevol

A cookiecutter template for the OrthoEvolution python package.

OrthoEvolution

OrthoEvolution is an easy to use and comprehensive python package which aids in the analysis and visualization of orthologous genes.

htseq-count-cluster

A python cli/wrapper for running multiple qsub jobs with HTSeq's htseq-count script on a SGE/SGI cluster.
